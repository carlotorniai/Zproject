Could you be a Data Scientist is a project developed by Carlo Torniai as final project for the Zipfian Academy program. It uses some machine learning techniques to classify an aspiring data Scientist profile and provide useful information such as characterization of their profiles, suggested skills, similar profiles etc.

You can find more details about the proejct in this blog post.
